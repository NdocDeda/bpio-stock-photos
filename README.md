# How to install

Open your terminal (CMD or Bash) and execute the next lines:
```bash
git clone https://github.com/NdocDeda/bpio-stock-photos.git && cd bpio-stock-photos.git
composer install
```

## How to run

After running the next line, open http://localhost:8000:
```bash
php artisan serve
```
